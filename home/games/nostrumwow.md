# NostrumWoW Connection & UI Guide

Welcome to the documentation page for **NostrumWoW**. This guide contains all the details necessary to get connected to the server, alongside the curated list of essential Wrath of the Lich King (WotLK 3.3.5a) addons used to optimize the gameplay experience.

---

## 🌐 How to Connect to NostrumWoW

To jump into the realm, follow these steps to properly configure your game client:

1. **Visit the Official Website**
   * Head over to [NostrumWoW](https://nostrumwow.com/) to register your game account.

2. **Locate Your Realmlist File**
   * Open your main **World of Warcraft** game directory.
   * Navigate to: `Data` ➔ `enUS` (or your specific language locale folder, such as `enGB`).

3. **Edit the Realmlist**
   * Open the `realmlist.wtf` file using a standard text editor like Notepad.
   * Delete any existing lines and replace them with the official NostrumWoW realmlist address:
     ```text
     set realmlist logon.nostrumwow.com
     ```
   * *Note: If you encounter an error when saving, right-click the file, select **Properties**, ensure that **Read-only** is unchecked, and try again.*

4. **Launch the Game**
   * Always start the game using **Wow.exe** in your main directory. Avoid using the default Blizzard launcher to prevent your configuration files from overwriting.
