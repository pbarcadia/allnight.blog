## 🌐 How to Connect to NostrumWoW 🧭

To jump into the realm, follow these steps to properly configure your game client:

1. **Visit the Official Website** 🏛️
   * Head over to [NostrumWoW](https://nostrumwow.com/) to register your account.

2. **Locate Your Realmlist File** 📁
   * Open your main **World of Warcraft** game directory.
   * Navigate to: `Data` ➔ `enUS` (or your specific language locale folder).

3. **Edit the Realmlist** ✍️
   * Open the `realmlist.wtf` file using a standard text editor like Notepad.
   * Delete any existing lines and replace them with the official NostrumWoW address:
```text
     set realmlist logon.nostrumwow.com
     ```
   * *Note: If you encounter a saving error, right-click the file, select **Properties**, ensure **Read-only** is unchecked, and save again.*

4. **Launch the Game** 🎮
   * Always start the game using **Wow.exe** in your main directory. Avoid using the default Blizzard launcher to prevent your configuration files from overwriting.
